# 📄 AI Resume Analyzer

🚀 A smart resume analysis web app that evaluates resumes and provides actionable feedback to improve job chances.

---

## 🌐 Live Demo
👉 https://resume-analyzer-hgavybbtjprtz8xpimtnsn.streamlit.app/

---

## 🧠 Features

- 📊 Resume scoring system
- ✅ Detects key sections (Projects, Skills, Experience)
- 📧 Extracts email and phone number
- 🔗 Detects LinkedIn profile
- 💡 Provides actionable feedback
- 🎯 Uses dynamic scoring logic
- 📁 Upload resume (.txt) or paste text
- 🎨 Clean and interactive UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Regex (Pattern Detection)

---

## ⚙️ How It Works

1. User uploads or pastes resume
2. System analyzes:
   - Structure
   - Keywords
   - Action words
3. Generates:
   - Score
   - Strengths
   - Areas to improve
